# Sales Dashboard

## Project Overview

This project analyzes retail sales performance using SQL and Power BI.

The objective is to identify key revenue drivers, profitable product categories, customer behavior, and regional performance to support business decision-making.

---

## Tools Used

- SQL (SQLite)
- Power BI
- Excel / CSV
- GitHub

---

## Dataset

Main fields:

- Sales

- Customer

- Segment

## Key Insights

1. West region generated the highest revenue.
2. Consumer segment contributed more than 50% of total sales.
3. Technology category produced the highest profit.
4. A small number of products generated a large portion of total revenue.
5. Revenue leaders were not always the most profitable products.
6. Top customers represented a significant share of overall revenue.
7. Technology generated the highest profit, while Furniture produced significantly lower profit despite strong sales.
